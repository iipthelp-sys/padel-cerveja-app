AIRTABLE SETUP

Padel Cerveja & Petiscos · Database Creator

1
Go to airtable.com/create/tokens, create a token with scopes: data.records:read, data.records:write, schema.bases:read, schema.bases:write and access to All bases.
2
Paste your token below and click Create My Database. This will create a new Airtable base called "Padel Cerveja" with all 4 tables fully configured.
3
Copy the Base ID and API Token shown after setup and paste them into the Padel app.
